# webARHIVEsite
A site where you can store active sites / сайт в котором можно хранить активные сайты
протестированно на ubuntu server 24 с использованием Nginx
для того чтобы загрузить свой сайт в архив сайтов нужно чтобы главный html файл вашего кода был с именем index и находился в папке имя которой является названием сайта и всё это в list/
ВАЖНО информация о сайтах в list/ хранится в data.json которые обновляются с помощью скрипта
